# 🧱 Stack — Stack Implementations in Python

**Author:** Prateek Waghmare  

---

## 📌 Overview
This folder contains **step-by-step implementations of the Stack data structure** in Python,  
ranging from beginner to intermediate level. Each file demonstrates the **LIFO (Last-In, First-Out)** principle  
using different approaches — from Python’s built-in list to a custom linked list structure.

All scripts are well-documented, numbered for learning order, and safe to execute individually.

---

## 📂 Files

- ⚙️ [01_stack_builtin_demo.py](01_stack_builtin_demo.py)  
  Demonstrates **basic stack operations** using Python’s built-in list.  
  *(Push, Pop, Peek, isEmpty, Size)*

- 🧩 [02_stack_class_demo.py](02_stack_class_demo.py)  
  Implements a **Stack class** internally using lists.  
  Covers object-oriented approach to Stack operations.

- 🔗 [03_stack_linkedlist_demo.py](03_stack_linkedlist_demo.py)  
  Implements Stack using a **singly linked list**.  
  Demonstrates manual node linking and memory-efficient LIFO behavior.

- 🧪 **practice_experiments/**  
  Reserved for **custom tests, mini-projects, and stack-based experiments**.

---

## 🧠 Concepts Covered
- Stack operations: `push()`, `pop()`, `peek()`, `is_empty()`  
- Internal implementation using:
  - Built-in lists (beginner)
  - Custom class (OOP)
  - Linked list (intermediate)
- Understanding of **LIFO principle**
- Practical usage demonstrations via example executions
